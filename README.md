# FinTrack

FinTrack is a full-stack **FinTech Expense Tracker & Budgeting System** designed to serve **individual users, small businesses, and large enterprises** through a powerful multi-module financial management platform. It provides intelligent tracking, budgeting, reporting, and financial insights using modern web technologies.

The system is built using **React** for the frontend, **Node.js** for the backend, and **MySQL** for secure and scalable data storage.
